# D365 → Odoo 17 Custom Modules / الوحدات المخصصة

## Overview / نظرة عامة

This package contains all 12 custom Odoo 17 modules developed for the
D365 to Odoo ERP migration project. All modules are bilingual (English / Arabic).

يحتوي هذا الحزمة على جميع الوحدات المخصصة الـ 12 لـ Odoo 17 التي طورتها
فريق مشروع ترحيل ERP من D365 إلى Odoo. جميع الوحدات ثنائية اللغة.

## Quick Start / البداية السريعة

### Prerequisites / المتطلبات
- Docker Desktop / Docker Engine + Docker Compose v2
- 4GB RAM minimum / 4 جيجابايت ذاكرة على الأقل

### Start Odoo / تشغيل Odoo

```bash
# Clone or extract this package
cd odoo_deployment

# Start services
docker compose up -d

# Wait ~60 seconds for first startup, then open:
# http://localhost:8069

# Create a new database, then install modules from:
# Settings > Apps > Search "c1" through "c13"
```

### Install Arabic Language / تثبيت اللغة العربية

```
Settings > Translations > Languages > Activate Arabic (ar_001)
Settings > Users > Your User > Language = Arabic
```

All modules include `i18n/ar.po` translation files that are loaded automatically
when Arabic is activated. / تتضمن جميع الوحدات ملفات الترجمة ar.po التي يتم تحميلها
تلقائياً عند تفعيل اللغة العربية.

## Configuration / التكوين

### C1 — PO Approval Matrix
`Purchase > Configuration > Approval Thresholds`
- Add tiers: Tier 1 (0–5,000), Tier 2 (5,001–50,000), Tier 3 (50,001+)
- Assign approvers to each tier

### C5 — Financial Dimensions  
`Accounting > Configuration > Financial Dimensions`
- Pre-loaded with sample: FIN, OPS, SAL (departments), NORTH, SOUTH, CTR (regions)
- Add project dimensions as needed

### C7 — Credit Bureau API
`Sales > Configuration > Credit Bureau`  
- Enter API URL, API Key, score thresholds

### C13 — Tax XML Export
`Accounting > Reporting > Tax Filing XML`
- Upload signing certificate as attachment in company settings first

## Arabic RTL Support / دعم العربية من اليمين لليسار

Odoo 17 natively supports Arabic RTL layout. Once Arabic language is activated:
- All module UI labels switch to Arabic translations
- Form views automatically apply RTL direction
- Reports render in Arabic with correct number formatting

يدعم Odoo 17 بشكل أصلي تخطيط العربية من اليمين لليسار. بمجرد تفعيل اللغة العربية:
- تتحول جميع تسميات واجهة المستخدم إلى الترجمات العربية
- تطبق طرق عرض النماذج تلقائياً اتجاه RTL

## Arabic as Default / العربية كلغة افتراضية

This package is configured with **Arabic as the default language**.

### What this means:
- `odoo.conf` sets `lang = ar_001` and `default_lang = ar_001`
- Docker Compose passes `--load-language=ar_001 --i18n-overwrite` on startup
- All module field labels are **Arabic-first** (English in parentheses)
- All email templates render in Arabic with RTL layout
- New users created will default to Arabic locale
- All `i18n/ar.po` files are included and auto-loaded

### To switch a user to English / للتبديل إلى اللغة الإنجليزية:
```
Settings > Users > [User] > Preferences > Language = English
```

### Module label order:
Every field string follows the pattern:
```python
string = "العربية / English"
```
When Arabic locale is active, Odoo displays the translation from `ar.po`
which maps to the pure Arabic string. English users see the combined label.

### Priority modules (fully Arabic-first enhanced):
1. **C1** — مصفوفة اعتماد أوامر الشراء — complete rewrite with AR-first labels,
   RTL email templates, Arabic chatter messages
2. **C5** — محرك الأبعاد المالية — dimension master and journal line fields
3. **C13** — تصدير ملف XML الضريبي — Arabic-first wizard
4. **C7** — واجهة برمجة مكتب الائتمان — Arabic alerts and banners
